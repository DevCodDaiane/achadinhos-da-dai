# Projeto: Achadinhos da Dai

## Descrição do Projeto
Este projeto é uma plataforma de curadoria de produtos da Shopee, com foco em moda e acessórios. A plataforma foi desenvolvida para organizar itens em categorias, como "Roupas", "Acessórios" e "Promoções", facilitando a navegação do usuário e direcionando-o para as páginas dos produtos na Shopee.

## Tecnologias Utilizadas
* **HTML:** Para a estrutura das páginas e a organização do conteúdo.
* **CSS:** Para a estilização visual e a criação de um design atraente e fácil de usar.

## Funcionalidades Principais
* **Curadoria de Produtos:** Seleção de itens específicos para o público-alvo.
* **Organização por Categorias:** Navegação intuitiva com itens divididos em categorias claras.
* **Links de Afiliados:** Direcionamento direto para as páginas de produtos da Shopee.
